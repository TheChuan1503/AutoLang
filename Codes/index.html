在<head>标签中填入
<script src="al.min.js"></script>

在所有标签之后，</body>标签之前填入
<script>
    al.setLangProp(语言配置对象 | JSON字符串 | 语言配置数组(详情此处))
    al.load(语言,国际化方式,回调函数)
</script>
或 
<script>
    al.httpGet("语言配置文件地址(同域)",function(r){
        al.setLangProp(r)
        al.load(语言,国际化方式,回调函数)
    })
</script>
al.setDefaultCountry({
    zh:"zh-CN",
    en:"en-US"
})
al.setDefault("zh-CN")
al.load()参数: 
语言:默认为系统语言
国际化方式:
    al.mode.HTML(默认):
        直接修改内部HTML
    al.mode.TEXT:
        直接修改内部文字(覆盖子元素)
    al.mode.REPLACE:
        替换元素内文字，作用有al属性元素，在元素内填入键，无法动态更改语言
回调函数:
    国际化完成后执行该函数(无参数)
al.setLangProp()参数: 
obj:
    配置对象 | url | 配置数组 | 字符串(json/yaml)
callback:
    加载配置后的回调函数，返回配置对象/数组
attr:
    {
        url:是否为url,
        yaml:是否为yaml
    }
在需要国际化的标签添加属性al="键"(al.mode.REPLACE无需键)
在语言配置中填入键对应的文字 
<span al="title"></span>
